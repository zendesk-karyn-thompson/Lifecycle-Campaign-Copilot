# HOW TO CREATE IN GOOGLE SLIDES:

## Slide 1 Setup:
1. Open Google Slides → Blank presentation
2. Delete existing text boxes
3. **Title text box:**
   - Text: "Campaign Planning Hub"
   - Font: Arial or Roboto
   - Size: 48pt
   - Bold
   - Position: Top center

4. **Subtitle text box:**
   - Text: "From Hours of Planning to Minutes of Impact"
   - Font: Arial
   - Size: 24pt
   - Position: Below title

5. **Left box (Problem):**
   - Insert → Shape → Rectangle
   - Add text: [Copy "THE PROBLEM" section above]
   - Font: Arial
   - Size: 14pt for bullets, 18pt for title
   - Black text, white background, thin border

6. **Right box (Solution):**
   - Insert → Shape → Rectangle
   - Add text: [Copy "THE SOLUTION" section above]
   - Font: Arial
   - Size: 14pt for bullets, 18pt for title
   - Black text, white background, thin border

7. **Footer:**
   - Small text box at bottom
   - Font: 10pt
   - Text: "Live Demo: zendesk-karyn-thompson.github.io/Campaign-Brainstorm"
